# 🚀 Project Workflow Dashboard

A detailed, intuitive dashboard for tracking collaborative projects and workflows.

## Features

✅ **Dashboard Overview**
- Real-time stats (total projects, in progress, completed, average progress)
- Quick filters by status
- Create new projects instantly
- Recent activity feed

✅ **Project Management**
- Detailed project cards with progress bars
- Status tracking (Planning, In Progress, Paused, Complete)
- Priority levels (Low, Medium, High)
- Task checklists with completion tracking
- Next action tracking
- Team member assignment
- Timeline & notes
- Fully editable via UI

✅ **Full CRUD Operations**
- Create projects
- Edit all project details
- Delete projects
- Add/remove/complete tasks
- Update progress in real-time

✅ **Responsive Design**
- Works on desktop, tablet, mobile
- Beautiful Tailwind CSS styling
- Navy, Teal, and gradient accents
- Smooth animations & transitions

## Tech Stack

- **Frontend**: Next.js 14, React 18, TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Database Ready**: Supabase integration (future)

## Getting Started

### Local Development

```bash
npm install
npm run dev
```

Visit http://localhost:3000

### Production Build

```bash
npm run build
npm start
```

## Deployment

### Deploy to Vercel (One Click)

1. Push to GitHub:
```bash
git remote add origin https://github.com/bpnelsen/project-workflow-dashboard.git
git branch -M main
git push -u origin main
```

2. Go to https://vercel.com
3. Click "New Project"
4. Connect your GitHub repository
5. Deploy!
